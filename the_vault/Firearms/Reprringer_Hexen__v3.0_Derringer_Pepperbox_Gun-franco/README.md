# Reprringer HEXEN (derringer pepperbox gun)

#### Author
franco

#### Version

3.0

#### Description 

So, latest reprringer, the Reprringer HEXEN (derringer pepperbox gun) by franco. Print white parts from ABS and red and yellow parts from PLA.

##### Parts:
* 3x M3x20 bolt
* 1x M3x25 bolt
* 1x M4x60 bolt
* 4x M3 nut
* 1x M4 self-locking nut
* 2x M3 washer
* 6x 8x1x50mm steel tube
* 1x some compression spring about 10x50mm
