#### Description
This is the CZ Scorpion Hand Stop Evo. There is a solid version and a light weight version. They use a M5 screw and nut to attach to any standard 1913 rail.  Both of these Hand Stops are inspired by the CZ Scorpion EVO Hand Stop.
