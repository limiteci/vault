# Ambi Safety Selector for the  AR15 and AR10

#### Author
unamed

#### Version
1.0

#### Description
Here we have a Ambidextrous Safety Selector for the AR-15 and AR-10.  This design was in part inspired by the BAD-ASS Ambi Safety Selector. The Levers are detachable and interchangeable on both sides for maximum flexibility and user preference.  There are a total of 9 Lever Designs and 81 Possible Lever Combinations.
