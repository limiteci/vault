# CZ Scorpion Evo 3 AK Style Safety Selector

#### Author 
unnamed and Sc0rp

#### Version
1.0


#### Description

This is an AK-styled safety selector for the CZ Scorpion EVO 3. This design was inspired by the "HB Industries AK style selector".

#### Print Instructions
* The file should be printed with the selector flat on the print bed surface. 
* If the hole doesn't print at proper size ream it open with a .108 bit or something slightly smaller than 1/8.
