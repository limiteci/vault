# Sisyphus Grip Adapter

#### Author 
WarFairy

#### Description
This is meant to make AR-15's California complaint. Please note, it has not been approved by the CA DOJ in any way shape or form. Until it has been, use at your own risk.  Included is a dummy buffer tube to attach a stock to the grip adapter if you do not have an extra actual buffer tube laying around. Dummy buffer tube is Mil-Spec.  Also included is the Threaded stock attachment which can be used as a stock and attached to the adapter.
