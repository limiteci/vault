# The Gryphon .22 Caliber Suppressor Baffle 

#### Author
Remove_Kebab

#### Version
1.0

#### Description
A.22 caliber monocore suppressor baffle design. It is 1 inch in diameter, and 8 inches long. The user should have a registered suppressor tube and follow all federal and state firearms laws before building this design.
