# CZ Scorpion Evo Pistol Triggers 

#### Author
Scorp

#### Version
1.0

#### Description

This is a trigger for the CZ Scorpion Evo Pistol. A flat trigger with grip grooves and a slightly curved flat trigger are also included in the STL folder. This design was inspired by Semper1371 CZ Scorpion Triggers.

#### Printing Instructions

* Print on side for maximum strength.
* Supports may be necessary for printing.
