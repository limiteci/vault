# 9mm Heckler & Koch USP 10 Round Magazine Baseplate

#### Author
Jelly

#### Version
1.0


#### Description

This is a 9mm Heckler & Koch USP Magazine Baseplate. It is designed to fit 10 round USP 9mm Magazines. Note: it does not add any capacity.

Succesfully test printed and fired with both designs. Long design for competition use or for comfort for those of us with very large hands. 

Printed with layer height of .3mm and rectilinear supports generated by Slic3r.
