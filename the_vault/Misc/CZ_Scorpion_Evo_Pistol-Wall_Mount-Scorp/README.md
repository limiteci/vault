# CZ Scorpion Evo Pistol Wall Mount

#### Author
Scorp

#### Version
1.0

#### Description
The  CZ Scorpion Evo Pistol Wall Mount is a way to mount a CZ Scorpion Evo to the wall using the mag well as the support.

For the next version, I'm making a universal base for it that you can swap out the mag so it can fit any firearm. 

#### Printing Instructions
* Print on side so the grain is vertical for maximum strength.
