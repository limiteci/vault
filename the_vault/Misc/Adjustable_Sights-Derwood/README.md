# Adjustable Sights

#### Author
Derwood

#### Version
1.0

####Description

These sights are adjustable and are designed to be mounted on a picatinny rail.

#### Non-printed parts list

* 4 #6 sheet metal screws cut to length for set screws. (grind end of screws flat with no sharp edges)
* 1 #6 x 1 1/4" for left/right adjustment
* 2 #6 x 1" for picatinny mount
