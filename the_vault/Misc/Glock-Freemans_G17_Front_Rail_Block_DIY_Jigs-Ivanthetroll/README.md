#### Description

This is the Freemans G17 Front Rail Block DIY Jigs. These jigs will aid people who aren't mechanically inclined in fabbing the rail blocks on their own. The first 3 jigs are for facing off - jigs 4 and 5 are for profiling. 

To keep this readme short, I'll upload a video showing how you use each jig, and how to use calipers (because you do still need a way to measure what you do.
